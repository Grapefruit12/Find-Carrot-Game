# Find-Carrot-Game
🥕시간안에 모든 당근을 찾는 게임

## [규칙]
1. 5초안에 5개 당근을 모두 찾아야 win!
2. 벌레 클릭시, 시간초과시 lose!

## 구현
1. 당근들과 벌레들이 랜덤으로 위치
2. 때에 맞게 음악 나오도록 구현
